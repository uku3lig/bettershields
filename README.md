**BetterShieldSounds** is a very simple mod aimed to fix sound inconsistencies with shields. Fixes [MC-105608](https://bugs.mojang.com/browse/MC-105068).

## Dependencies
* [ukulib](https://modrinth.com/mod/ukulib)
